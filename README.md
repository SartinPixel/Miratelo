# Miratelo - Triheroes
This is a game project I'm working on. Triheroes is set to be an 3d adventure Zelda-Roguelike game currently made with unity 2020.
This repository contains only source code (no assets) and is currently public just **for fun and feedbacks** during the early development of my game project.
Please **do not copy, redistribute, or reuse** the core codes, just take inspiration from it if it is useful for you.
