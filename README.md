# Miratelo - Triheroes
<br> This is a game project I'm working on. Triheroes is set to be an 3d adventure Zelda-like game, currently made with unity 2020.
<br> This repository contains only source code and is currently public just **for fun and feedbacks** during the early development of my game project.

Please **do not redistribute** the core codes
## **project:**
##### core - layer that executes the state machines and modules on top of unity
##### code -  all modules for this game specific gameplays
##### script - module trees including player controller, NPC AI, and in game event
##### content - game assets

all modules names follow snake_case naming, suffixes:
<br> ac : actor controller action
<br> c : controller
<br> m : module data
<br> s : module system
<br> d : gameobject-less module
<br> pc : player controller module`
